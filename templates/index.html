How it works

From blank canvas to
exported site in four steps.

01

Pick a starting point

Start from a generated template, pull one from a URL, or open a blank canvas. All three routes land you in the same editor.

Drag components onto the page, arrange them freely, and customize styling without writing CSS. Switch between desktop and mobile view as you go.

Use the code generator to build what you need. Hit a wall? Ask the AI chatbot. Fix errors with the error solver. Everything inside the same platform.
